VII Visualisation des frequences sonore avec FFT

Pour analyser et générer des forme qui bouge selon le son pour cela nous devons : 

Prendre la variable audioSource précédément declarer et y intégrer = audioContext.createMediaElementSource(audio1) // Pour déclarer une source audio depuis la variable audio1 contenant le code base64 pour y créer un noeud audio en JS ( truc compliquer a voir ) 

Ensuite je prend la variable analyser = audioContext.createAnalyseur(); // Pour exposer les donnée audio de temps de fréquences que nous utiliserons plus tard pour notre visualiseur.  

Ensuite je connecte audioSource a analyser  pour exposer la synchronisation et les fréquence sur ce fichier sous forme d'objet de donnée.  

Ensuite je connecte analyser avec audioCtx.destination (périphérique audio par default doncprobablement les hauts parleur du PC )

Ensuite avec analyzer.fftSize une propriété spéciale qui reprensente le nombre d'échantillons audio que nous voulons dans notre fichier de donnée ( 248 est la valeur par default mais nous pouvons mettre une des ces valeurs 32, 64 .... 32768 ) Plus le nombre sera éleve plus nous aurons de bar qui analyse le son dessiner sur le canvas 

Je créer une constante bufferLenght pour la mémoire tampon = analyser.frequencyBinCount; // Pour convertir dans un format spéciale
( La valeur buffer.Lenght = la moitié de fftSize ) et il represente le nombre de bar 

Ensuite je créer une variable dataArray = Uint8Array(bufferLenght) pour récupérer les donnée formaté envoyer par mon analyseur dans un type de tableau Uint8 ( Peut cotenir seulement des entier 8bits )
